# Python Fundamentals

This repository contains core Python practice programs covering:

- Loops (for, while)
- Functions
- Object-Oriented Programming (OOPS)
- Exception Handling
- File Handling (coming soon)

## Files Included

- loops.py – Demonstrates for and while loops
- functions.py – Covers function definitions and return values
- oops.py – Basic class and object implementation
- exception_handling.py – Try, except and error handling examples

## Purpose

This repository is created to strengthen Python fundamentals and build a strong backend development foundation.
